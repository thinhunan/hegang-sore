# 第五章 财报分析方法论

## 财报五大核心内容

| 部分 | 作用 | 阅读要点 |
|------|------|---------|
| 致股东信 | 管理层理念与愿景 | 看创始人视野、思考深度 |
| 财务报表 | 核心（三张表） | 必须会看 |
| 报表附注 | 数据详细说明 | 会计政策、项目明细 |
| 审计报告 | 可信度背书 | 谁审的 + 结论（无保留意见最佳） |
| 公司报告 | 股东权益、治理、社会责任 | 无重大异常可看可不看 |

**风险提示**：公司自己提示的风险要当回事，半年内可能兑现。

**彼得·林奇两句话**：
1. 永远不要投资不了解财务状况的公司
2. 仔细研究财报，确认公司不会破产

## 财报三维度分析

**逻辑**：看权益定家底 → 看经营定赚钱能力 → 看现金定生存能力。

### 第一维度：看权益（资产负债表）

**公式**：资产 - 负债 = 所有者权益（净资产）

**资产**：
- 流动资产：现金、应收款、存货（应收款过多 = 回款问题）
- 非流动资产：厂房、设备、无形资产（品牌类公司无形资产价值大）

**负债**：区分短期/长期负债，关注长期负债对股东权益的压力。债务展期不减少总负债。

**股东权益三部分**：
- 注册资本（注意虚高）
- 资本公积（未来增资扩股）
- 未分配利润（持续分红能力）

**结论**：股东权益越小越不值得关注；接近零 → 抛弃；负数 → 坚决不买。

### 第二维度：看经营（利润表）

**收入**：
- 主营业务分布、占比、增长、第二增长曲线（巴菲特第一关键指标）
- 大额存款的利息收入（苹果、茅台）

**成本费用**：
- 成本：刚性运营成本
- 三费：销售费用、管理费用、财务费用
- **毛利率** = (营业收入 - 营业成本) / 营业收入 × 100%

**巴菲特标准**：同行业中毛利率高的公司是好公司。白酒/游戏部分公司毛利率 80-90%。

### 第三维度：看现金（现金流量表）

| 类型 | 含义 | 健康信号 |
|------|------|---------|
| 经营活动现金流 | 日常经营"吃饭钱" | 持续正现金流（茅台、苹果） |
| 投资活动现金流 | 投资方向 | 长期大额流出无回报 = 陷阱 |
| 筹资活动现金流 | 融资能力 | 区分股权融资 vs 刚性负债 |

**核心原则：现金为王**。权益大、利润好但现金流走弱 → 公司可能走向衰退。

**警惕**：IPO 后账上现金多但经营现金流为负 = 造血能力不足。

# 使用示例

## 示例 1：分析一家公司

**用户**：用何刚方法论分析一下宁德时代值不值得投资

**Agent 流程**：
1. 读取 ch01（扇面模型）→ 判断当前政策/宏观环境
2. 读取 ch02（何三点）→ 分析动力电池行业
3. 读取 ch03 + company-score-sheet → 三维度量化打分
4. 拉取最新财报/行情数据
5. 输出 investment-analysis-report 格式报告

**输出要点**：
- 行业：千亿级+、上游主导（电池为核心）、龙头竞争
- 公司打分：规模性 5、竞争性 5、方向感 4、稳定性 4、盈利性 5、风控力 4 → 均分 4.5/5 ≈ 9
- 结论：具备价值投资关注价值，但需关注估值和竞争格局变化

**用户**：半导体行业现在还能投吗？用何三点分析

**Agent 流程**：
1. ch01 扇面模型快速过宏观/政策面
2. ch02 何三点深入分析
3. 标注数据时效，获取最新行业数据

**输出结构**：
```
## 何三点分析：半导体

### 行业规模
- 级别：千亿级+
- 趋势：AI 驱动持续增长

### 产业链条
- 类型：上游主导（设备/设计/制造）
- 价值点：光刻机、芯片设计、代工

### 竞争格局
- 垄断+充分竞争并存
- 龙头：中芯国际、北方华创等
- 搅局者筛选三问...
```

## 示例 3：选基金

**用户**：帮我用何刚基金三步法选一只稳健的债券基金

**Agent 流程**：
1. ch04 三步法
2. 第一步：求稳型 → 偏债/纯债
3. 第二步：长期 → 中大规模 10-50 亿
4. 第三步：经理 5-10 年从业、年化收益稳定

## 示例 4：读财报

**用户**：帮我用何刚的方法读一下茅台最新财报

**Agent 流程**：
1. ch05 五部分框架定位重点
2. 三维度：权益（家底）→ 经营（毛利率）→ 现金（经营现金流）
3. 关注：未分配利润、毛利率、经营现金流充沛度

## 示例 5：黄金配置

**用户**：现在黄金还能买吗？

**Agent 流程**：
1. ch06 四大需求因素
2. 重点评估投资需求 + 央行储备需求
3. 关联宏观：美元、利率、地缘风险
4. 输出配置逻辑，不给买卖指令

## 示例 6：完整投资决策

**用户**：我想投资新能源赛道，帮我做个全面分析

**Agent 流程**（完整扇面模型）：
1. **政策面**：双碳政策、新能源补贴/税收
2. **宏观面**：GDP、CPI/PMI、利率
3. **基本面-行业**：何三点分析新能源
4. **基本面-公司**：推荐用评分表对比 2-3 家龙头
5. 输出完整 investment-analysis-report
